FLOOD PREDICTION USING MACHINE LEARNING

## OVERVIEW
This project predicts the probability of floods using environmental and geographical features with a Machine Learning model.

## TECHNOLOGIES USED
1.Python
2. Pandas
3.NumPy
4.Scikit-learn
5.Google Colab

## MACHINE LEARNING MODEL
Random Forest Regressor

## PROJECT OVERFLOW
1. Data Loading
2. Data Preprocessing
3. Train-Test Split
4. Model Training
5. Prediction
6. Model Evaluation


## OUTPUT
The model predicts flood probability based on environmental factors.
